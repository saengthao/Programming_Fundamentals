Personal Workout Program


PROBLEM STATEMENT
Create an algorithm that asks the user for their fitness goal and level, recommends a workout based on their
input, and generates a fitness plan for a specified number of days. The plan alternates between workout days
and rest days and displays the full schedule in the browser.
  
Nouns: Algorithm, Fitness Goal, Workout, Plan, Days, Schedule, Browser
  
Verbs: Create, Ask, Recommend, Generate, Display



DEFINING DIAGRAM
INPUTS:
  Fitness goal
  Fitness level
  Number of days for the fitness plan
  
PROCESSING:
  Prompt the user for their fitness goal (lose weight, gain muscle, or improve endurance).
  Prompt the user for their fitness level (beginner, intermediate, advanced).
  Recommend a workout based on the fitness goal and level using criteria.
  Prompt the user for the number of days for the fitness plan.
  Generate a fitness plan switching between workout days and rest days using a loop.
  Store and format the generated plan.
  
OUTPUTS:
  Recommended workout based on the fitness goal and level.
  A structured fitness plan displaying workout and rest days for the specified number of days.



SOLUTION ALGORITHM
personalizeFitnessProgram
  Prompt user for fitnessGoal.
  Prompt user for fitnessLevel.
	
  Determine recommended workout based on fitnessGoal and fitnessLevel using predefined criteria.
  Prompt user for the number of days, numDays.
  Initialize counter to 1.
	
  WHILE counter <= numDays
  IF counter is odd
    Add "Day counter: (Recommended workout)" to the fitness plan.
  ELSE:
    Add "Day counter: Rest Day" to the fitness plan.
    Increment counter by 1.
  ENDWHILE
	
  Output fitnessGoal, fitnessLevel, and the generated fitness plan message.
END
